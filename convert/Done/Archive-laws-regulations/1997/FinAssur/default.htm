Financial Assurances Enforcement

Description:  This rulemaking defined the scope and responsibilities of CIWMB staff in pursuing appropriate enforcement actions.

Affected Regulatory Code Sections:  Title 27, California Code of Regulations (CCR), Division 2, , Subdivision 1, Chapter 6, by adopting a new Subchapter 4, and a new Article 1, sections 22270 through 22278.

History:  Approved by OAL and filed with Secretary of State on June 3, 1997.  Became effective on July 7, 1997.

Rulemaking Documents:  

Notice of Proposed Action (Word 6.0/95, 25KB)
Information published in the California Regulatory Notice Register 45 days prior to the end of the public comment period which references the proceedings, statutory authority, an informative digest describing existing laws, any mandates and the effect of the proposed action on small business.

Initial Statement of Reasons (Word 6.0/95, 38KB)
A document containing a description of the condition or problem that this regulation is intended to address and the rationale for the determination that this regulation is necessary to carry out the purpose for which it is proposed. Any alternatives considered and technical documents are also noted here.

An easily understood description of the proposed text.
