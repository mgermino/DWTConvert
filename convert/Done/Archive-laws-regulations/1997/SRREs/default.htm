Source Reduction and Recycling Elements (SRRE)

Description:  The regulations provide guideline and content requirements for the Source Reduction and Recycling Elements (SRREs).

Affected Regulatory Code Sections:  Title 14, California Code of Regulations (CCR), Division 7, Chapter 9, Article 6.2, Sections 18730 - 18748.

History:  These regulations were approved by OAL February 13, 1997 and filed with the Secretary of State on that date.
