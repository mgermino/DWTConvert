Sustainable (Green) Building

Tools for Local Governments

Many local governments have expressed an interest in starting a residential green building program. The following tools and resources are provided for governments to benefit from the work already developed by State agencies, local agencies, and other private and nonprofit entities.

Existing Programs

Many local jurisdictions have developed various green building programs. Many of these existing programs have focused on public construction, but their information may add some insight into getting your program started.
