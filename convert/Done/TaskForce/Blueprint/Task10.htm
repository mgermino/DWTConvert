Sustainable (Green) Building

Blueprint Tasks: Partnerships

Below is Action Item 10 and associated tasks from the strategy, Building Better Buildings: A Blueprint for Sustainable State Facilities (Blueprint). The Blueprint contains a ten-point plan to implement the Governor's sustainable building goal. Links to key deliverables are provided, if available.
